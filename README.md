SIAKAD MINI

Deskripsi

SIAKAD MINI adalah aplikasi web sederhana berbasis PHP yang dibuat menggunakan konsep Object Oriented Programming (OOP).
Aplikasi ini digunakan untuk mengelola data akademik seperti mahasiswa, mata kuliah, nilai, dan Kartu Hasil Studi (KHS).

⸻

Fitur
	•	Data Mahasiswa
	•	Data Mata Kuliah
	•	Input Nilai
	•	Tampilan KHS
	•	Perhitungan IPK

⸻

⸻

Cara Menjalankan
	1.	Install XAMPP atau Laragon
	2.	Letakkan folder project di dalam htdocs
	3.	Jalankan Apache
	4.	Buka browser dan akses:
http://localhost/siakad-mini
